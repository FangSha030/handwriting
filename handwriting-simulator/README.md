# 手写笔迹模拟器 (Handwriting Simulator)

基于 GitHub 开源手写生成项目（如 HandwritingGenerator）的架构思路进行二次开发与改进，专注于**后期处理微调**与**界面优化**，目标是模拟出更加真实自然的手写笔迹效果。

## 核心特性

### 任务要求 1：手写后期微调与横格对齐

- **框选编辑**：在预览画布上通过鼠标拖拽框选需要修改的文字或区域
- **连续批量操作**：支持连续多次框选和修改，所有微调操作累积进行，最后统一预览（取消"框选一次即保存"的低效交互）
- **高精度微调**：对框选文字支持位置（0.5px 步长）、字号（1% 步长）、旋转（0.5° 步长）的极细腻调节
- **边界包容**：框选判定范围包含字符边界框的留白区域，确保选中判定准确
- **编辑态实时显示打印体**：微调操作时所有文字显示为标准打印体（思源宋体/黑体），降低视觉干扰
- **最终态渲染手写体**：仅在点击"最终预览"时，根据调整后的排版参数渲染真实手写体效果
- **横格线自动检测**：根据文本布局自动生成横格线，支持均匀生成模式
- **自动基线对齐**：将文字基线吸附到横格线，容差 30px，保证文字与横格自然贴合

### 任务要求 2：优化操作界面 UI

- **核心画布主导**：采用"左侧栏 + 中间大画布 + 右侧栏"布局，画布作为视觉和操作核心
- **功能模块分区**：文本输入与参数配置区（左）、画布预览与微调区（中）、操作与导出区（顶/右）
- **悬浮工具栏**：框选文字后在选中区域附近自动弹出，包含位置微移、字号微调、旋转、确认/取消
- **高精度控制面板**：右侧栏提供精确数值输入，支持键盘输入微小数值弥补鼠标拖拽误差
- **网格与辅助线**：编辑态可开关对齐网格、字符边界框、自动基线对齐辅助线
- **现代极简护眼风格**：扁平化设计，低饱和度护眼配色（米白纸张色 + 墨绿强调色）
- **状态模式视觉切换**：编辑态（暖黄徽章 + 辅助线 + 打印体）↔ 最终态（墨绿徽章 + 手写体）
- **实时提示与引导**：Toast 通知 + 底部操作提示，降低学习成本
- **流畅渲染**：基于 Canvas 2D + DPR 适配，拖拽框选跟手无卡顿

## 使用指南

### 基本流程

1. **输入文本**：在左侧"文本输入"区输入或粘贴需要模拟手写的文字（支持中英文混排）
2. **配置参数**：在"手写参数"区选择手写字体、字号、行高、字间距、墨水颜色、笔迹自然度
3. **框选微调**（编辑态）：
   - 在画布上鼠标拖拽框选需要调整的文字
   - 悬浮工具栏自动弹出，可进行位置微移、字号微调、旋转
   - 也可在右侧"高精度微调"面板输入精确数值
   - 点击 ✓ 确认微调（可继续框选其他区域，所有微调累积）
4. **横格对齐**（可选）：切换到右侧"横格对齐"标签，自动检测横格线并基线对齐
5. **最终预览**：点击顶部"最终预览"按钮，切换到手写体渲染
6. **导出**：点击"导出图片"保存 PNG，或"导出配置"保存 JSON 参数

### 快捷操作

| 操作 | 说明 |
|------|------|
| 鼠标拖拽 | 在画布上框选文字 |
| `Esc` | 取消当前选区 |
| 悬浮工具栏 ✓ | 确认当前微调（可继续框选） |
| 悬浮工具栏 ✗ | 取消选区 |
| 顶部"编辑态" | 切换回编辑模式（打印体 + 辅助线） |
| 顶部"最终预览" | 切换到手写体渲染 |

## 开发参照说明

本项目参考了 GitHub 上成熟手写生成开源项目的核心架构思路：

- **逐字符布局 + 随机扰动**：参考 HandwritingGenerator 等项目的核心算法，对每个字符施加位置、旋转、缩放、不透明度的随机扰动
- **种子随机数**：保证同一文本多次渲染结果一致，便于微调后保持效果

在此基础上增强的部分（二次开发重点）：

1. **编辑态/最终态分离**：编辑时显示打印体便于精准框选，最终才渲染手写体
2. **框选微调系统**：支持连续批量框选、累积微调、悬浮工具栏 + 高精度数值面板双通道调整
3. **横格线检测与基线对齐**：自动检测横格线并将文字基线吸附对齐
4. **现代护眼 UI**：三栏画布主导布局、低饱和度配色、模式视觉切换
