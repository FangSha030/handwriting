# 手写笔迹模拟器 Pro (Handwriting Simulator Pro)

基于 GitHub 开源手写生成项目（[cpython666/HandwritingGenerator](https://github.com/cpython666/HandwritingGenerator)）与 [beautifulcarrot](https://beautifulcarrot.com/v2/quick_mode/) 的架构思路进行二次开发与全面增强，专注于**后期处理微调**、**界面优化**、**背景图导入**、**自动文本流**与**多页排版**，目标是模拟出更加真实自然的手写笔迹效果。

## 核心特性

### 一、手写后期微调与横格对齐（任务要求1）

- **框选编辑**：在预览画布上通过鼠标拖拽框选需要修改的文字或区域
- **连续批量操作**：支持连续多次框选和修改，所有微调操作累积进行，最后统一预览
- **高精度微调**：对框选文字支持位置（0.5px 步长）、字号（1% 步长）、旋转（0.5° 步长）的极细腻调节
- **边界包容**：框选判定范围包含字符边界框的留白区域
- **编辑态实时显示打印体**：微调操作时所有文字显示为标准打印体（思源宋体/黑体）
- **最终态渲染手写体**：仅在点击"最终预览"时渲染真实手写体效果
- **横格线自动检测**：根据文本布局自动生成横格线，或从背景图检测横格线
- **自动基线对齐**：将文字基线吸附到横格线，容差可调

### 二、优化操作界面 UI（任务要求2）

- **核心画布主导**：采用"左侧栏 + 中间大画布 + 右侧栏"布局
- **功能模块分区**：文本输入与参数配置区（左）、画布预览与微调区（中）、操作与导出区（顶/右）
- **悬浮工具栏**：框选文字后在选中区域附近自动弹出
- **高精度控制面板**：右侧栏提供精确数值输入
- **网格与辅助线**：编辑态可开关对齐网格、字符边界框、基线辅助线、可写区域
- **现代极简护眼风格**：扁平化设计，低饱和度护眼配色
- **状态模式视觉切换**：编辑态（暖黄徽章 + 辅助线 + 打印体）↔ 最终态（墨绿徽章 + 手写体）
- **实时提示与引导**：Toast 通知 + 底部操作提示

### 三、背景图导入与管理（对齐 HandwritingGenerator + beautifulcarrot）

- **内置纸张模板**：横线纸、方格纸、点阵纸、田字格、拼音格、空白纸 6 种模板
- **背景图上传**：支持上传任意图片作为书写背景（信纸、作业本、稿纸等）
- **文件夹批量导入**：支持选择文件夹批量导入多张背景图
- **背景图管理**：左侧栏显示背景图列表与缩略图，可切换/删除
- **多背景轮换**：多页排版时自动轮换使用多张背景图

### 四、自动文本流与多页排版（核心增强）

- **自动分页**：输入 5000 字但只有 1 张背景图时，自动将剩余文字分配到新页面
- **可写区域检测**：自动分析背景图像素，检测白色/浅色可书写区域
- **横格线检测**：从背景图自动检测横格线位置，文字基线对齐
- **多区域支持**：一张背景图可检测多个可写区域（如多栏排版）
- **页面导航**：底部翻页控件，支持前后翻页查看
- **批量导出**：一键导出所有页面 PNG

```bash

npm run build

```

## 使用指南

### 基本流程

1. **输入文本**：在左侧"文本"标签输入文字（支持中英文混排，支持 5000+ 字自动分页）
2. **选择背景**：切换到"背景"标签
   - 选择内置纸张模板（横线/方格/田字格等）
   - 或上传自定义背景图（信纸、作业本扫描件等）
   - 或批量导入文件夹中的多张背景图
3. **选择字体**：切换到"字体"标签，选择 14 种手写字体之一，或上传自定义 TTF
4. **配置参数**：在右侧"手写"标签调整字号、行高、笔迹粗细、抖动等
5. **框选微调**（编辑态）：鼠标拖拽框选文字，悬浮工具栏微调
6. **横格对齐**：右侧"横格"标签，自动检测横格线并基线对齐
7. **最终预览**：点击顶部"最终预览"切换到手写体渲染
8. **导出**：导出当前页 PNG / 导出全部页 PNG / 导出配置 JSON

### 自动文本流示例

**场景**：输入 5000 字，但只有 1 张背景图

1. 上传 1 张横线纸背景图
2. 输入 5000 字文本
3. 系统自动检测背景图可写区域与横格线
4. 文字自动填充到第 1 页可写区域
5. 超出部分自动分配到第 2、3、4... 页（复用同一背景图）
6. 底部显示"第 X/N 页"，可翻页查看
7. 点击"导出全部"一键导出所有页面
